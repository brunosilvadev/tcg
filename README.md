# Pindorama

**Pindorama** is a browser-based virtual trading card game themed around native South American mythology, deities, cultures, and rituals. The name comes from the Tupi word for Brazil — "land of palm trees" — evoking the rich indigenous heritage that inspired the game.

## What is it?

Players log in, build their collection by opening virtual booster packs, and discover cards featuring AI-generated art and lore rooted in indigenous South American traditions. The initial collection focuses on Tupi culture, with future expansions planned for other native peoples of the Americas.

## Core Loop

1. User signs in via Google
2. User earns points by completing simple daily tasks (e.g. daily login, ad interactions)
3. Points unlock booster packs
4. Opening a pack is a rewarding, animated experience
5. Cards are added to the user's collection
6. Missing cards appear as shadowed silhouettes, encouraging completion

## Tech Stack

- **Frontend:** Angular
- **Backend:** .NET Web API (C#)
- **Database:** PostgreSQL
- **Auth:** Google OAuth 2.0
- **Hosting:** Azure
- **Card Art:** AI-generated images
